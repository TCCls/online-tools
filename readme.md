# 在线多种编码解码工具
1. 对字符串或文件进行编码，或者对已加密的字符进行解密显示为明文
2. Base64等编码比较通用，可以解决中文乱码时,对中文进行不同方式的编码可有效避免中文乱码
3. 本站上传在ipfs上，不会记录您的任何信息,请放心使用
